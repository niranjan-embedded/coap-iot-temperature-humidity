# CoAP-Based IoT Temperature & Humidity Monitoring

## Overview
This project implements a lightweight IoT server using the CoAP protocol for remote monitoring of temperature and humidity. It is designed for resource-constrained embedded systems.

## Features
- Real-time temperature and humidity monitoring
- Lightweight CoAP protocol over UDP
- Efficient communication for embedded devices

## Technologies Used
- Embedded C
- CoAP Protocol (UDP)
- Microcontroller (Beaglebone Black)
- UART Communication

## Working
Sensor data is collected from the device and transmitted using the CoAP protocol to a server. The server processes the data and allows remote monitoring.
